# significance-testing — 显著性检验

## 用途
对实验结果做统计显著性检验,从假设设定、检验方法选择、显著性水平确定到前提假设检查,产出完整的检验报告(含 p 值、效应量、结论),避免"只报 p 值"的不严谨做法。

适用于:两组方法对比、多方法对比、效应显著性判断、结果稳健性验证等场景。

## 触发条件
- 用户要求"显著性检验""p 值""效应量"时。
- 需要判断本方法 vs baseline 的差异是否统计显著时。
- 用户提到"t 检验""ANOVA""Wilcoxon""显著性"时。

## 操作步骤
1. **定假设**:明确零假设 H0(如两组无差异)和备择假设 H1(如两组有差异),定方向性(双尾 / 单尾)。
2. **选检验方法**:按数据特性和样本量选(t 检验 / Wilcoxon / Mann-Whitney / ANOVA / Kruskal-Wallis),说明选择理由。
3. **定显著性水平**:通常 α=0.05,多重比较时做校正(Bonferroni / Holm / FDR)。
4. **执行检验**:运行检验,记录 p 值、检验统计量。
5. **报告 p 值和效应量**:除 p 值外必须报告效应量(Cohen's d / η² / Cliff's delta)和置信区间。
6. **检查前提假设**:正态性(Shapiro)、方差齐性(Levene)、独立性,前提不满足时换非参数方法。

## 调用示例
```python
from scipy import stats
import pingouin as pg

# 两组对比(正态)
t, p = stats.ttest_ind(group_a, group_b)
d = pg.compute_effsize(group_a, group_b, eftype='cohen')

# 非参数(非正态)
stat, p = stats.mannwhitneyu(group_a, group_b)

# 多组对比 + post-hoc
f, p = stats.f_oneway(g1, g2, g3)
```

## 输出格式
显著性检验报告,含:
- H0 / H1 假设
- 检验方法及选择理由
- 显著性水平 α(及是否做多重比较校正)
- p 值 / 检验统计量 / 效应量 / 置信区间
- 前提假设检查结果(正态性 / 方差齐性 / 独立性)
- 结论(显著 / 不显著 + 效应大小解读)

## 约束
- **样本不足时说"不足以支撑显著性结论"**:不得用"趋势显著"等模糊措辞掩盖功效不足。
- 必须报告效应量,不只报 p 值。
- 前提假设不满足时必须换非参数方法,不得强行用参数检验。
- 多重比较必须校正,否则结果标"未校正,应谨慎解读"。
