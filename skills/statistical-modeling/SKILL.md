# statistical-modeling — 统计建模

## 用途
对实验数据做统计建模,选择匹配数据类型的统计模型,完成拟合和模型诊断,产出可解释的统计模型报告,为实验结论提供统计支撑。

适用于:实验数据分布拟合、变量关系建模、效应估计、统计推断等场景。

## 触发条件
- 用户要求"统计建模""拟合模型""数据分析"时。
- 实验数据需要做统计建模推断时。
- 用户提到"回归""分布拟合""统计模型"时。

## 操作步骤
1. **探索数据分布**:画直方图 / Q-Q 图,做正态性检验(Shapiro-Wilk),判断数据类型(连续 / 计数 / 分类型)。
2. **选统计模型**:按数据类型和变量关系选模型(线性回归 / 广义线性模型 / 混合效应模型 / 生存分析等),说明选择理由。
3. **拟合**:用对应方法拟合模型(OLS / MLE / 贝叶斯),记录估计参数和拟合过程。
4. **诊断**:做残差分析、异方差检验、多重共线性检查(VIF)、过拟合检查,逐项标注是否通过。
5. **输出模型报告**:汇总模型形式、参数估计、拟合度(R²/AIC/BIC)、诊断结论。

## 调用示例
```python
import statsmodels.api as sm
import scipy.stats as stats

# 正态性检验
stat, p = stats.shapiro(data)
print(f"Shapiro p={p:.4f}")

# 线性回归拟合
X = sm.add_constant(X)
model = sm.OLS(y, X).fit()
print(model.summary())

# 残差诊断
residuals = model.resid
stats.probplot(residuals, dist="norm", plot=plt)
```

## 输出格式
统计模型报告,含:
- 数据类型和分布特征
- 选用的模型及选择理由
- 参数估计(系数 / 标准误 / 置信区间)
- 拟合度指标(R² / AIC / BIC / 对数似然)
- 模型诊断结果(残差 / 异方差 / 共线性 / 过拟合)

## 约束
- **模型选择必须匹配数据类型**:连续型用线性模型,计数型用泊松/负二项,分类型用 logistic,不得乱套。
- 必须做模型诊断,诊断不通过不得使用模型结论。
- 不得为迎合结论事后换模型。
- 多重比较必须做校正。
