# citation-integration — 引用整合

## 用途
把引用整合进正文:识别需要引用的位置、匹配文献、插入引用标记、生成参考文献列表、检查引用一致性。确保每条引用都经过工具核查(citation-verification),不编造引用。

适用于:论文/报告/申报书正文起草后,引用整合环节。

## 触发条件
- 正文初稿完成,需要插入引用
- 用户要求"加引用""补参考文献"
- 已有引用但需要核查一致性
- citation-verification 报告产出后,需要修正引用

## 操作步骤

### 步骤 1:识别需引用位置
扫描正文,识别需要引用的位置:
1. **事实陈述**:引用他人结论或数据(如"X 方法在 Y 上达到 SOTA")
2. **方法借用**:用了别人的方法/组件(如"我们采用 [Smith 2023] 的损失函数")
3. **对比对象**:与已有工作对比(如"与 Wang 2024 不同,我们...")
4. **背景介绍**:Introduction 和 Related Work 中的领域陈述
5. **定义引用**:用了别人提出的概念/术语

不需要引用的位置:
- 自己的方法描述(本研究的)
- 自己的实验结果
- 公知常识(如"深度学习是机器学习的子领域")

### 步骤 2:匹配文献
1. 从文献库(BibTeX)检索匹配的文献
2. 匹配策略:
   - 按关键词匹配(方法名/任务名/数据集名)
   - 按语义匹配(陈述的内容与文献摘要比对)
3. 找不到匹配文献的位置,标注 `[待补引用:需要引用什么样的工作]`
4. 不得编造引用,不得"看起来该有引用就编一个"

### 步骤 3:插入引用标记
1. 按目标格式插入引用标记:
   - 数字式:[1] / [Smith2023]
   - 作者-年:(Smith, 2023) / Smith (2023)
   - LaTeX:\cite{key} / \citep{key} / \citet{key}
2. 引用密度:
   - Introduction:密集(每段 3-5 处)
   - Related Work:最密集(每句 1-2 处)
   - Method:适度(借用的方法处)
   - Results:少量(对比时)
   - Conclusion:几乎不引用
3. 多文献引用用 [1, 2, 3] 或 (Smith 2023; Wang 2024)

### 步骤 4:生成参考文献列表
1. 收集正文所有引用的文献
2. 按目标格式排序:
   - 数字式:按正文出现顺序
   - 作者-年:按第一作者字母序
3. 生成 BibTeX 文件或直接列表
4. 检查:
   - 正文引用的都进了列表
   - 列表里的都在正文被引用(未引用的清理)
5. 调用 citation-verification 核查每条引用

### 步骤 5:检查引用一致性
1. **标记-列表一致**:正文每个标记都能在列表找到,反之亦然
2. **元数据一致**:正文写的(Smith 2023)与列表里 Smith 的年份一致
3. **数字一致**:正文引用的数字与该文献报告的数字一致
4. **格式一致**:全文引用格式统一(不混用 [1] 和 (Smith, 2023))
5. **自引控制**:自引不超过 20%

## 调用示例
```
正文段落:
  "LoRA [待补引用] 通过低秩分解显著减少了可训练参数。"
匹配文献:Hu et al., 2021, "LoRA: Low-Rank Adaptation of Large Language Models", arXiv
插入标记:
  "LoRA \cite{hu2021lora} 通过低秩分解显著减少了可训练参数。"
BibTeX:
  @article{hu2021lora,
    title={LoRA: Low-Rank Adaptation of Large Language Models},
    author={Hu, Edward J. and others},
    journal={arXiv preprint},
    year={2021}
  }
核查:调用 citation-verification 确认存在性 + 元数据
```

## 输出格式
- **带引用的正文**:正文 + 引用标记
- **参考文献列表**(references.bib):BibTeX 格式
- **引用一致性报告**:
  - 正文标记数 / 列表条目数
  - 孤儿标记(正文有,列表无)
  - 孤儿条目(列表有,正文无引用)
  - 元数据不一致项
  - 自引比例
- **待补引用清单**:每条注明位置 + 需要什么样的文献

## 约束
- 不编造引用,找不到匹配的标 `[待补引用]`
- 每条引用必须经过 citation-verification 核查
- 正文标记和参考文献列表必须一一对应
- 引用格式全文统一,不混用数字式和作者-年
- 自引控制不超过 20%
- Related Work 不得漏掉关键工作(由文献研究员确认覆盖度)
- 待补引用在投稿前必须补全或显式标注
