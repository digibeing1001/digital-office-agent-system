# conflict-of-interest — 利益冲突检查

## 用途
检查论文是否存在利益冲突,核查资助来源、作者关系、商业利益,生成利益冲突声明,确保有利益冲突时必须声明。

适用于:论文投稿前利益冲突核查、资助来源审查、商业利益声明等场景。

## 触发条件
- 用户要求"检查利益冲突""COI 检查""利益冲突声明"时。
- 论文投稿前需要核查利益冲突时。
- 用户提到"利益冲突""conflict of interest""资助"时。

## 操作步骤
1. **检查资助来源**:列出所有资助来源(基金 / 企业 / 机构),判断资助方是否与研究结论有利益关联。
2. **检查作者关系**:核查作者是否与相关企业/机构有雇佣 / 顾问 / 持股关系,是否可能影响研究客观性。
3. **检查商业利益**:研究方法/产品是否涉及商业利益(专利 / 创业公司 / 商业产品),是否可能因结论获益。
4. **生成声明**:无论有无利益冲突,生成利益冲突声明(有则声明具体内容,无则声明"无利益冲突")。

## 调用示例
```
输入:论文草稿 + 资助信息 + 作者关系信息
输出:利益冲突报告 + 声明文本
```
```markdown
## 资助来源
1. 国家自然科学基金(编号 XXX)— 无商业利益
2. 某公司合作项目 — ⚠️ 资助方产品在本研究中评估
## 作者关系
- 作者 A:无相关商业关系
- 作者 B:⚠️ 为某公司顾问,该公司产品在本研究中对比
## 商业利益
- 研究涉及的方法已申请专利(作者 A 为发明人)

## 利益冲突声明(建议文本)
"本研究受某公司合作项目资助。作者 B 为该公司顾问。
所评估方法已申请专利。作者声明上述关系未影响研究客观性。"
```

## 输出格式
利益冲突报告,含:
- 资助来源清单及利益关联判断
- 作者关系清单及利益关联判断
- 商业利益清单
- 利益冲突声明文本(可直接放入论文)

## 约束
- **有利益冲突必须声明**:不得隐瞒资助来源或作者商业关系。
- 即便"无利益冲突"也要生成声明文本(期刊要求)。
- 资助方产品在研究中评估时必须声明,不得回避。
- 专利 / 创业公司等商业利益必须披露。
