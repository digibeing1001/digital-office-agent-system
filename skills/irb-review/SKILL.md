# irb-review — IRB审查

## 用途
当研究涉及人类或动物被试时,做伦理委员会(IRB)审查,检查 IRB 批准、知情同意、风险收益比,确保涉及人类被试的研究有 IRB 批准。

适用于:涉及人类被试的研究、涉及动物实验的研究、伦理审查等场景。

## 触发条件
- 用户要求"IRB 审查""伦理审查""检查伦理"时。
- 研究涉及人类被试(问卷调查 / 用户实验 / 临床数据)或动物实验时。
- 用户提到"IRB""伦理委员会""知情同意"时。

## 操作步骤
1. **识别是否涉及人类/动物被试**:判断研究是否涉及人类被试(问卷调查 / 用户实验 / 个人数据采集)或动物实验,若无则直接标"不适用"。
2. **检查 IRB 批准**:若涉及,检查是否有 IRB(或同等伦理委员会)批准文件,记录批准编号和日期。
3. **检查知情同意**:人类被试是否签署知情同意书,知情同意是否包含必要要素(目的 / 风险 / 自愿 / 退出权)。
4. **检查风险收益比**:评估研究对被试的风险与预期收益是否合理,风险是否已最小化。
5. **输出 IRB 审查报告**:汇总以上检查,给出伦理合规判断。

## 调用示例
```
输入:研究方案 + IRB 批准文件(若有)+ 知情同意书(若有)
输出:IRB 审查报告
```
```markdown
## 是否涉及人类/动物被试:是(人类被试,用户实验)
## IRB 批准
- 状态:✅ 已获 IRB 批准
- 编号:IRB-2024-XXX
- 日期:2024-XX-XX
## 知情同意
- 状态:⚠️ 知情同意书未提及退出权
- 建议:补充"被试可随时退出且无不良后果"
## 风险收益比
- 风险:低(问卷+行为测试)
- 收益:中等(科学贡献)
- 判断:合理
```

## 输出格式
IRB 审查报告,含:
- 是否涉及人类/动物被试
- IRB 批准状态(编号 / 日期 / 批准机构)
- 知情同意检查结果
- 风险收益比评估
- 总体伦理合规判断

## 约束
- **涉及人类被试必须有 IRB 批准**:无 IRB 批准直接标"不合规",不得发表。
- 知情同意必须完整,缺要素标"待改"。
- 弱势群体(儿童 / 孕妇 / 患病人群)需额外保护措施。
- 风险收益比不合理的研究标"不合规"。
